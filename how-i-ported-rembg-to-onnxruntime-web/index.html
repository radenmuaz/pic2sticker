How I Ported RemBG to ONNXRuntime Web

March 23, 2026 • 10 min read

Bringing professional-grade background removal directly to the browser required breaking down Python's rembg library and porting its logic to WebAssembly. This is a comprehensive, step-by-step developer tutorial on how NoBG achieves native-like performance purely on the client. You can view the full source code for our implementation here: https://github.com/radenmuaz/pic2sticker.

Step 1: The Neural Network (ONNX Runtime Web)

We export the PyTorch U2Net model into the ONNX format (specifically the lightweight u2netp.onnx). Using onnxruntime-web, we can execute this model natively in the browser leveraging WebAssembly (WASM) and WebGL execution providers.

Step 2: A Minimal Reproducible Example

Running ONNX in the browser is surprisingly elegant. If you simply want to see the u2netp.onnx model generate a raw probability mask, you can replicate this entire process in a single HTML file with inline JavaScript manipulating an HTML5 canvas directly without any bundlers required.

Step 3: The Post-Processing Problem

The code gives you a soft probability mask, but the neural network alone isn't enough for professional graphics. The Python rembg library heavily relies on Alpha Matting, Gaussian Blurring, and Morphological Opening (via OpenCV and pymatting) to accurately refine tricky edges.

Step 4: Compiling OpenCV to WebAssembly

To replicate the exact Python cv2 environment, we use Emscripten (emsdk) to compile C++ OpenCV directly to WebAssembly, shrinking it from 30MB down to static WASM core libraries.

Step 5: Writing the C++ Wrapper

With our static libraries built, we wrote a custom C++ file (postprocess.cpp) that effectively bridges the raw inference mask from the ONNX JavaScript execution into OpenCV for matrix thresholds and trimaps.

Step 6: Frontend React Integration

Finally, inside our Vite application environment (src/lib/rembg.ts), we dynamically load postprocess.wasm only when needed, passing the RAW ONNX mask straight into C++ memory for precise and rapid alpha matting.
